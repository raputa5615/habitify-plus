# Habitify+ 요구사항 (1편 버전)

## 한줄 설명

사용자가 매일 습관을 체크하고, 연속 일수를 보면서 동기를 얻는 정적 웹사이트.

## 1편에서 만들 기능

- 습관을 입력하는 칸
- 입력된 습관들의 목록
- 각 습관마다 "오늘 완료" 체크 박스
- 연속 일수(streak) 표시
- 데이터는 브라우저에 저장 (localStorage)

## 사용할 기술

- HTML, CSS, 자바스크립트만 사용
- 외부 라이브러리 최소화
- 빌드 도구 없음 (브라우저에서 바로 실행)

## 디자인 방향

- 단순함 우선
- 흰 배경, 검은 글씨, 포인트 색 하나만
- 모바일에서 잘 보여야 함

## 제약 (1편에서 하지 않는 것)

- 회원가입 / 로그인 (2편에서)
- 데이터베이스 (2편에서)
- AI 기능 (2편에서)
